## Usage

```shell
npm i usd-query
```

```ts
import $ from 'usd-query'

$('.foo') // document.querySelector('.foo')
```

```ts
import { $$ } from 'usd-query'

$$('.foo') // document.querySelectorAll('.foo')
```
